### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 首页、4. 论坛、5. 论坛详情、6. 公告栏、7. 新闻、8. 新闻详情、9. 疫苗信息、10. 疫苗详情、11. 预约计划、12. 个人中心

管理员功能：1. 登录、2. 轮播图、3. 公告栏、4. 用户管理、5. 论坛列表管理、6. 论坛分类列表管理、7. 新闻列表管理、8. 新闻分类列表管理、9. 接种者信息、10. 疫苗信息、11. 预约疫苗、12. 入库疫苗、13. 预约计划、14. 疫苗出库、15. 接种记录
